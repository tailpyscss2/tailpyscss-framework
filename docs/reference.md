# TailPySCSS API Reference

Comprehensive reference for all utility classes generated by the framework.

## üì¶ Flexbox & Display

| Class | CSS Property |
| :--- | :--- |
| `.flex` | `display: flex` |
| `.grid` | `display: grid` |
| `.hidden` | `display: none` |
| `.block` | `display: block` |

## ‚ñ¶ Grid

| Class | CSS Property |
| :--- | :--- |
| `.grid-cols-{n}` | `grid-template-columns: repeat(n, 1fr)` |
| `.col-span-{n}` | `grid-column: span n / span n` |

## üìè Sizing

| Class | Description |
| :--- | :--- |
| `.w-{size}` | Width (e.g. `w-full` = 100%, `w-1/2` = 50%) |
| `.h-{size}` | Height (e.g. `h-screen` = 100vh, `h-full` = 100%) |
| `.max-w-{size}` | Max Width |
| `.min-h-screen` | `min-height: 100vh` |

## üìç Position

| Class | Properties |
| :--- | :--- |

| `.top-0`, `.right-0`, `.bottom-0`, `.left-0` | Zero offsets |
| `.z-{index}` | z-index (0, 10, 20... 50) |

## üé® Colors

| Class | Description |
| :--- | :--- |
| `.text-{color}` | Sets text color |

| `.border-{color}` | Sets border color |
