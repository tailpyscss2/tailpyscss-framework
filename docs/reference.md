# TailPySCSS Class Reference

This document lists all the utility classes generated by TailPySCSS.

## üé® Colors

Generated from `theme.colors` in `tailpy.config.json`.
*Replace `{color}` with your color name (e.g., `primary`, `red`, `white`).*

| Class | CSS Property | Description |
| :--- | :--- | :--- |
| `.text-{color}` | `color` | Sets text color |
| `.bg-{color}` | `background-color` | Sets background color |
| `.border-{color}` | `border-color` | Sets border color |
| `.hover\:bg-{color}` | `background-color` | Sets background on **hover** |
| `.hover\:text-{color}` | `color` | Sets text color on **hover** |

## üìè Spacing

Generated from `theme.spacing` in `tailpy.config.json`.
*Replace `{size}` with keys like `0`, `1`, `4`, `8`, etc.*

### Padding
| Class | CSS Property |
| :--- | :--- |
| `.p-{size}` | `padding` (all sides) |
| `.pt-{size}` | `padding-top` |
| `.pr-{size}` | `padding-right` |
| `.pb-{size}` | `padding-bottom` |
| `.pl-{size}` | `padding-left` |
| `.px-{size}` | `padding-left` & `padding-right` |
| `.py-{size}` | `padding-top` & `padding-bottom` |

### Margin
| Class | CSS Property |
| :--- | :--- |
| `.m-{size}` | `margin` (all sides) |
| `.mt-{size}` | `margin-top` |
| `.mr-{size}` | `margin-right` |
| `.mb-{size}` | `margin-bottom` |
| `.ml-{size}` | `margin-left` |
| `.mx-{size}` | `margin-left` & `margin-right` |
| `.my-{size}` | `margin-top` & `margin-bottom` |

## üìê Layout & Flexbox

### Display
`.block`, `.inline-block`, `.inline`, `.flex`, `.inline-flex`, `.hidden`, `.grid`

### Flex Container
| Class | Description |
| :--- | :--- |
| `.flex-row` / `.flex-col` | Direction |
| `.flex-wrap` / `.flex-nowrap` | Wrapping |
| `.justify-start`, `.justify-center`, `.justify-end`, `.justify-between` | Justify Content |
| `.items-start`, `.items-center`, `.items-end`, `.items-stretch` | Align Items |

### Flex Items
| Class | Description |
| :--- | :--- |
| `.flex-1` | Grow and shrink |
| `.flex-none` | Prevent growing/shrinking |
| `.grow` | Allow growing |

### Grid System
| Class | Description |
| :--- | :--- |
| `.grid-cols-1` to `.grid-cols-12` | Define number of columns |
| `.gap-{size}` | Grid/Flex gap |
| `.gap-x-{size}` | Column gap |
| `.gap-y-{size}` | Row gap |

### Sizing (Width & Height)
**Width**:
- `.w-full` (100%), `.w-screen` (100vw)
- Fractions: `.w-1/2`, `.w-1/3`, `.w-2/3`, `.w-1/4`, `.w-3/4`
- Content: `.w-min`, `.w-max`

**Height**:
- `.h-full` (100%), `.h-screen` (100vh)

### Positioning
*   **Position**: `.static`, `.fixed`, `.absolute`, `.relative`, `.sticky`
*   **Z-Index**: `.z-0`, `.z-10`, `.z-20`, `.z-30`, `.z-40`, `.z-50`

## üöÄ Usage with @apply

You can use any of these classes in your SCSS files:

```scss
.my-card {
    @apply bg-white p-6 rounded-lg shadow-lg;
    @apply flex flex-col items-center;
}
```

## Arbitrary Values (JIT)

TailPySCSS supports arbitrary values for many utilities. Use the `-[value]` syntax to apply custom styles on the fly without changing your config.

| Category | Prefix | Example Usage | Output CSS |
| :--- | :--- | :--- | :--- |
| **Width** | `w-[...]` | `@apply w-[350px];` | `width: 350px;` |
| **Height** | `h-[...]` | `@apply h-[100vh];` | `height: 100vh;` |
| **Min Height** | `min-h-[...]` | `@apply min-h-[80rem];` | `min-height: 80rem;` |
| **Padding** | `p-[...]`, `px-[...]` | `@apply px-[20px];` | `padding-left: 20px; padding-right: 20px;` |
| **Margin** | `m-[...]`, `mx-[...]` | `@apply mt-[5em];` | `margin-top: 5em;` |
| **Position** | `top-[...]`, `left-[...]` | `@apply top-[50%];` | `top: 50%;` |
| **Z-Index** | `z-[...]` | `@apply z-[9999];` | `z-index: 9999;` |
| **Border Radius** | `rounded-[...]` | `@apply rounded-[15px];` | `border-radius: 15px;` |
